# 课题：纸卷动物
年级：四年级
单元：纸艺造型

## 教学过程

### 导入：看纸卷像什么
教师出示纸卷和几个动物图片，请学生猜一猜纸卷可以变成什么动物身体。

### 探究：发现卷、折、剪的关系
学生观察纸卷作品，找出身体、耳朵、尾巴分别用了什么纸艺方法。

### 示范：连接身体和局部
教师示范卷纸、剪耳朵、粘贴尾巴，提醒胶水位置和结构稳定。

### 创作：完成纸卷动物
学生选择一种动物完成纸卷造型，并尝试添加表情或纹样。

### 展评：说出最像的地方
学生展示作品，说出自己最满意的动物特征。
