课题：《足下生辉》旧鞋改造设计课
年级：三年级
单元：第五单元 足下生辉
本课承接上一课学生完成的旧鞋设计草图。学生需要带来一只旧鞋、上一课设计草图和可用于改造的旧材料。
本课不是重新画一双鞋，而是把已有草图变成真实旧鞋作品。
教学目标：
1. 学生能对照上一课设计草图，说出旧鞋要改造的部位和制作路径。
2. 学生能根据设计意图选择旧材料，完成旧鞋作品的主体制作。
3. 学生能用5句话设计故事说明作品对象、解决的问题、材料方法、满意处和改进想法。
教学准备：
学生准备旧鞋、设计草图、旧材料、剪刀、胶水、彩纸、布条、纽扣等。
教师准备旧鞋改造示例、工具安全提示、新鞋发布会展示牌。
教学过程：
1. 回顾草图与任务确认 5分钟：学生拿出上一课设计草图，说明今天准备把旧鞋改造成什么。
2. 制作路径讨论 8分钟：教师组织学生比较粘贴、包裹、添加、绘制等制作路径，确认材料如何服务设计意图。
3. 旧鞋作品制作 22分钟：学生根据草图改造旧鞋，教师巡视材料连接、结构稳定和工具安全。
4. 5句话设计故事 7分钟：学生围绕作品对象、解决的问题、材料方法、满意处和改进处写下设计故事。
5. 新鞋发布会准备 3分钟：学生整理作品和故事卡，准备下一环节展示发布。
评价证据：
学生是否带来并使用草图；作品是否体现旧鞋改造；材料选择是否服务设计意图；5句话设计故事是否能说明作品；发布会是否有作品和故事卡。
